# WeatherWidgetX
View weather and air quality index infomation on the menu bar.
Currently only support weather and AQI in Changsha, China.

![Screenshot](https://ws3.sinaimg.cn/large/006tNc79ly1fha52b9gaij30im0j8gqj.jpg)

## Build Requirement
- Xcode 9.0+
- Swift 4.0+

## Install Requirement
- macOS 10.10+
